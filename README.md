# Stock Selection ML

Sistema de Machine Learning para seleção de ações da B3 com potencial de valorização superior a 5% em 5 pregões.

## Tecnologias

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost

## Pipeline

1. Coleta de dados
2. Feature Engineering
3. Definição do Target
4. Purged Time Series Cross Validation
5. Treinamento
6. Avaliação

## Features

- RSI
- MACD
- Momentum
- Volatilidade
- Alpha
- Beta
- Distância da máxima de 52 semanas

## Resultados

- Random Forest
- XGBoost
- XGBoost Tunado
